The AI Movie Matcher I created is a personalized recommendation engine that simplifies how users find films. In a world where too many choices can overwhelm us, this tool uses a mathematical method to suggest movies from both Hollywood and Bollywood based on a user's specific genre interests. It connects global cinema with local favorites like Dhurandar and 3 Idiots, as well as any other Hollywood movie. 

The Intelligence Behind the Product
The core engine uses the Jaccard Similarity Index, a data science algorithm that measures the overlap between two sets. By comparing the user's inputs with our movie database, the system calculates an exact match percentage. This ensures that recommendations are based on logic instead of random choice, providing a clear "Match Score" for every suggestion. 

User-Centric Design 
Designed with the user in mind, the application has a strong text-processing layer. It automatically manages capitalization and extra spaces, ensuring that the AI remains effective even if the user makes typing mistakes. The output features a visual star-rating system, making it easy for users to spot their top matches at a glance.

Technical Implementation This project is built using Pure Python, showing that effective AI solutions don’t always need complex libraries or setups. It emphasizes clean code, efficient data structures like Dictionaries and Sets, and an interactive Command Line Interface (CLI) that anyone with a basic Python environment can access.
